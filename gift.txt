// --- Úloha 2: Gaussovský filter (spracovanie obrázkov) ---

::U2-Q1 Gaussian blur paralelizácia:: Ktorý TBB konštrukt je najvhodnejší na paralelizáciu 2D konvolúcie obrázka? {
    =tbb\:\:parallel_for s blocked_range2d # Správne — 2D dekompozícia prirodzene mapuje na blokové rozdelenie obrázka.
    ~tbb\:\:parallel_reduce # Redukcia nie je potrebná pri nezávislom výpočte pixelov.
    ~tbb\:\:parallel_pipeline # Pipeline je pre prúdové spracovanie, nie pre dátový paralelizmus.
    ~tbb\:\:task_group # Príliš nízkoúrovňové pre túto úlohu.
}

::U2-Q2 Gaussian blur závislosť dát:: Prečo je Gaussovský blur vhodný na paralelizáciu? {
    =Výpočet každého výstupného pixelu je nezávislý od ostatných výstupných pixelov. # Správne — ide o embarrassingly parallel problém.
    ~Jadro filtra je symetrické. # Symetria jadra nesúvisí s paralelizovateľnosťou.
    ~Obrázok je uložený v pamäti ako 1D pole. # Spôsob uloženia neovplyvňuje nezávislosť výpočtov.
    ~Filter používa iba sčítanie a násobenie. # Typ operácií nie je rozhodujúci faktor.
}

// --- Úloha 3: Producer-Consumer Pipeline ---

::U3-Q1 Pipeline synchronizácia:: Prečo je v TBB parallel_pipeline potrebná synchronizácia pri aktualizácii zdieľaných štatistík? {
    =Viaceré vlákna môžu súčasne pristupovať k rovnakej pamäti a spôsobiť race condition. # Správne — bez mutexu by sa dáta mohli poškodiť.
    ~Pipeline filtre bežia vždy sekvenčne. # Nie — paralelné filtre bežia súčasne.
    ~TBB automaticky zabezpečuje thread-safety pre všetky premenné. # Nie — TBB synchronizuje len vlastné kontajnery.
    ~Mutex zrýchľuje výpočet. # Nie — mutex pridáva réžiu kvôli zamykaniu.
}

::U3-Q2 TBB filter modes:: Aký je rozdiel medzi serial_in_order a parallel filtrom v tbb\:\:parallel_pipeline? {
    =serial_in_order spracováva položky postupne v poradí, parallel ich spracováva súčasne viacerými vláknami. # Správne.
    ~serial_in_order je rýchlejší než parallel. # Nie — serial filter je obmedzujúci faktor.
    ~parallel filter zachováva poradie položiek. # Nie — paralelný filter nezaručuje poradie.
    ~Oba filtre fungujú rovnako, líšia sa len menom. # Nie — majú zásadne odlišné správanie.
}
